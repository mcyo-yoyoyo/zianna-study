# 伴学精灵 · 第一步：环境搭建与基础架构

## 技术选型

- **Vite 5** + **React 18** + **React Router 6**
- **Tailwind CSS 3**（马卡罗扩展色在 `tailwind.config.js`）
- 视口：`< 768px` 为**家长端**，`≥ 768px` 为**孩子端**（`AppContext` + `matchMedia`）

## 本地运行

前提：本机已安装 **Node.js 18+**（建议 LTS）。

在工程根目录执行：

```bash
cd "D:\AI Project\Zianna Study"
npm install
npm run dev
```

开发服务器**优先**使用 `http://localhost:5175/`。若 5175 已被占用，Vite 会顺延到 5176、5177… **请务必看终端里 `➜ Local:` 的那一行** 用浏览器打开。

**打不开时排查：**

1. 在工程目录先执行并保持窗口不关：`Set-Location "D:\AI Project\Zianna Study"; npm run dev`
2. 先试 **`http://127.0.0.1:5175/`**（与 `localhost` 等价，少数环境下更稳）
3. 用 **Chrome / Edge**；地址以 **http** 开头，不是 https
4. 若报「端口被占用」又不需要旧进程：任务管理器结束对应 **Node.js**，再重新 `npm run dev`

- **如何验证响应式**：开发者工具中切换手机/宽屏，或直接拖拽窗口宽度，顶部提示条会切换「家长/孩子」视图。

## 常用命令

| 命令 | 说明 |
| --- | --- |
| `npm run dev` | 开发服务器，热更新 |
| `npm run build` | 生产构建，输出到 `dist/` |
| `npm run preview` | 本地预览构建结果 |

## 目录说明（与 `提示词.md` 对齐）

- `src/pages/`：路由级页面 `Home` / `Spelling` / `ErrorBook` / `Dashboard`
- `src/components/`：公共与业务子组件（后续步骤逐步填充）
- `src/layouts/RootLayout.jsx`：全站布局（头、底栏、内容区）
- `src/contexts/AppContext.jsx`：全局面板角色（家长/孩子）
